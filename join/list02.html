회원가입

  1. 현재단계 1단계 약관동의
  2. 2단계 본인인증
  3. 3단계 회원정보입력
  4. 4단계 가입완료

회원정보입력

  • * 는 필수 입력 항목이므로 반드시 기입해주시기 바랍니다.
  • 비밀번호는 영대문자(A-Z, 26개), 영소문자(a-z, 26개), 숫자(0-9, 10개), 특수문자(22개) 중 3개 종류 이상으로 구성해야 합니다.
  • 특수기호중에 10개, -, \', [,], ., -, <, >는 입력하실 수 없습니다. 입력 시 자동 삭제됩니다.

아이디 및 패스워드

  • 영문(소문자) 또는 영문(소문자)와 숫자 조합하여 5~24자로 입력하세요.

  • 영문(대/소문자), 숫자, 특수문자 중 3가지 이상의 종류로 조합하여 9~24자 이내로 입력하세요.
    (단, -, \', [,], ., -, <, > 제외)

  • 비밀번호를 다시 한 번 입력해주세요.

개인회원정보

  • 성명 *필수 입력 항목
  • 성별 *필수 입력 항목
  • 생년월일 *필수 입력 항목
  • 휴대전화번호 *필수 입력 항목
  • 이메일 *필수 입력 항목
    @

수신여부 설정

  • 휴대전화 문자서비스 *필수 입력 항목

    휴대전화 문자 서비스를 받으시겠습니까?

    * 비밀번호 찾기 시 입력된 휴대전화번호로 임시 비밀번호를 발급해드립니다. 휴대전화문자서비스에 동의하셔야 서비스를 이용하실 수 있습니다.
